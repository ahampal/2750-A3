Description:
Web GUI for vCard Parser. Front end client makes use of bootstrap and communicates with the server
using ajax. Back end server written in javascript and makes function calls to parser library written
in C

Compilation:
-ensure node modules are installed
-cd to parser by typing 'cd parser' (ensure you are already in project directory)
-Create dynamic library by typing 'make parserDynamic' into terminal
-Run server by typing npm run dev 'portNum'

Assumptions and limitations:
-No known limitations
